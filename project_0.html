Description

In this assignment, you will create:
  1. Point3 Class that supports operations:
    • Normalization
    • Dot Product
    • Cross Product
  2. Mat3 Transformations that supports operations:
    • Multiplication with other Mat3 matrices
    • Multiplication with Point3 values
    • Inversion
    • Transpose
    • Helper function for translations
    • Helper function for scalings
    • Helper function for rotations
    • Helper function for viewport transformation
  3. Canvas Renderer which:
    • Maps Normalized Device Coordinates (NDC) to the left view screen.
    • Clips points outside the viewport space
    • Draws Lines
    • Draws Points
  4. WebGL Renderer which:
    • Maps Normalized Device Coordinates (NDC) to the left view screen.
    • Uses glBegin(...), glEnd(...), glVertex2f(...), glColor3f(...) commands
    • Draws Lines
    • Draws Points
  5. Feature support:
    • Zoom in control (slider)
    • Arbitrary Transformation Support
    • Color Range Support
